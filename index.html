circles

by kieran

This app works best with device motion data. Click the button below to allow access:

circles is an interactive parametric art piece that explores how a simple parametric drawing algorithm can create wildly different images. users are invited to interact with the piece using touch, motion, and traditional peripherals to see how it reacts to changing parameters.

DRAG

👆🏽↔️ 1 finger left/right
increase/decrease circle radius
👆🏽↕️ 1 finger up/down
increase/decrease disturbance amplitude
✌🏽↔️ 2 fingers left/right
increase/decrease spacing between circles
✌🏽↕️ 2 fingers up/down
increase/decrease margin

MOVE

move your device to make the circles rotate faster

TAP

👆🏽👆🏽👆🏽 3 fingers
change colors
👆🏽👆🏽👆🏽👆🏽 4 fingers
open control panel

SCROLL

scroll in/out
increase/decrease circle radius
hold z + scroll in/out
increase/decrease spacing between circles
hold x + scroll in/out
increase/decrease disturbance amplitude
hold c + scroll in/out
increase/decrease margin
hold c + scroll in/out
increase/decrease shape tightness

MOVE

move your cursor to make the circles rotate faster

BUTTONS

spacebar
change colors
escape
open control panel

save image
f
save image
+
increase number of points per circle
-
