circles

by kieran

This app works best with device motion data. Click the button below to allow access:

circles is an interactive parametric art piece that explores how a simple parametric drawing algorithm can create wildly different images. users are invited to interact with the piece using touch, motion, and traditional peripherals to see how it reacts to changing parameters.
